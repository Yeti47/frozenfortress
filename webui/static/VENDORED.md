# Vendored third-party assets

All third-party assets in `webui/static/` and `webui/.tools/` are pinned to the
versions listed below and verified by SHA-256 before use, per the project
[AGENTS.md](../../AGENTS.md) dependency-pinning policy.

To re-vendor an asset:
1. Update the version below.
2. Re-download with the documented command.
3. Update the SHA-256 in this file.
4. Commit the asset, this file, and any code changes in one commit.
